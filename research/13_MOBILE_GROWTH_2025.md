# Mobile-First Growth & PWA Strategy 2025

## Key Findings

### PWA Best Practices for Crypto Apps
- **Service Workers** for offline covenant viewing
- **Web Push Notifications** via Push Protocol for task updates
- **Add to Home Screen** with custom icons and splash screens
- **Background Sync** for transaction status polling
- **Share API** integration for covenant link virality

### Mobile Wallet Connection
- **RainbowKit Mobile** supports WalletConnect v2 with deep linking
- **Coinbase Smart Wallet** offers passkey-based mobile login
- **Dynamic.xyz** provides the smoothest email → wallet flow on mobile

### Social Growth Mechanics
- **Referral codes** with on-chain attribution (ERC-721 referral NFTs)
- **Guild.xyz** role-gated communities for covenant creators
- **Farcaster Frames** for embedded covenant creation/sharing
- **Push Protocol** for wallet-native notifications
